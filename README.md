# FIFA World Cup 2026 Dashboard

A live match dashboard for the 2026 FIFA World Cup. Track live scores, upcoming fixtures, recent results, group stage standings, and the knockout bracket — all in one place.

## Features

- **Live scores** — in-play matches are highlighted with a real-time progress bar and auto-refresh every 60 seconds
- **Upcoming fixtures** — all scheduled matches with countdown timers for games starting within 4 hours
- **Recent results** — finished matches with final scores and winner highlights
- **Group standings** — full group stage tables including form guides (last 5 matches)
- **Knockout bracket** — visual bracket view that updates as teams advance
- **Team search** — filter any view by team name

## Data

Match data is sourced from [football-data.org](https://www.football-data.org) and refreshes every 60 seconds during live matches. There is an upstream delay of approximately 5–10 minutes on live score updates due to the data provider's free tier.
